Vim 使用入门

发表于 2018-05-27 | 阅读次数:
字数统计: 1,252

本章主要整理终端里输入vimtutor后的教程的基本操作，供刚开始使用Vim的同学参考和学习。

第一讲

  1. 在vim里面可以使用hjkl键来代替鼠标以及箭头键来移动光标。当熟悉hjkl的操作之后，效率将会大大提升。h(左移), j(下移), k(上移), l(右移)
  2. 从shell中进入vim编辑器只需要使用 vim <filename> 就可以了
  3. 在vim中有两种模式：正常模式和编辑模式。正常模式下是不允许添加文本的，编辑模式才允许添加文本。如果进入了编辑模式，我们可以按下 <ESC> 键来返回到正常模式。要注意在编辑模式中是无法使用hjkl键来移动光标的
  4. 在正常模式中可以通过 x 键来删除光标右边的字符
  5. 有两种方法进入编辑模式：
    • 敲 i 键在光标初开始插入文本
    • 敲 A 键在光标所在行后添加文本
  6. 当编辑完文本以后，回到正常模式，然后就可以退出Vim编辑器回到shell了，这里又有两种方式：
    • :q! 放弃所有的改动退出
    • :wq 保存改动退出

第二讲

本讲主要针对删除操作

  1. 删除当前光标之后的一个单词 dw
  2. 删除当前光标之后的所有内容 d$
  3. 删除整一行的内容 dd
  4. 正常模式下的命令的格式：operator [number] motion. 可以通过使用number来重复操作，比如 d2w 就是删除两个单词
  5. 移动光标到行首 0；移动光标到行末$
  6. 想要撤销前一次操作使用 u；撤销对这一行的所有操作 U；撤销之前的撤销命令，恢复以前的操作结果: CTRL-R

第三讲

本讲主要针对置入操作

  1. 当我们删除了某个文本内容的时候，该文本内容会被放在寄存器中。然后使用 p 可以将寄存器的内容放在光标之后。如果最后一次删除的是一整行，那么该行会被放置于光标所在行的下一行
  2. 要替换光标所在位置的字符，请输入 r 和要替换掉原位置字符的新字符即可
  3. 当我们需要改变一个区间内的字符，可以使用 c 命令。可以更改c命令。使用 c [number] motion ，比如 c2w 就是修改两个单词，c$就是修改当前光标到句末的内容

第四讲

本讲主要针对定位、搜索、替换操作

  1. 按 CTRL-G 会显示当前光标所在行位置，记住改行的行号有助于跳转。然而其实代码文件在配置过以后是直接显示行号的
  2. 输入大写 G 会跳转到文件的最后一行；输入 gg 会跳转到文件的第一行；输入一个行号，然后再输入 G 可以直接跳转到该行
  3. 按下 / ，然后输入要搜索的字符串，可以往下搜索包含该字符串的所在行；用 ?的话就是往上搜索。继续往相同方向寻找相同串的话按 n，往逆向方向找的话就是按 N，这里的方向是针对一开始是使用 / 还是 ? 来说的。回退到光标的上一个位置使用 CTRL-O
  4. 当我们要找到某个括号的匹配的时候，使用 %
  5. 当我们要批量替换字符的时候：
    • 替换光标所在行的第一个匹配的单词: :s/old_string/new_string
    • 替换光标所在行的全部匹配的单词: :s/old_string/new_string/g
    • 替换多行的全部匹配单词：:首行号,尾行好s/old_string/new_string/g
    • 替换整个文件中的全部匹配单词: :%s/old_string/new_string/g

第五讲

本讲主要讲怎样在Vim里面调用外部linux指令，比如ls

  1. :!command 用于执行一个外部命令
  2. :w filenme 用于将VIM当前正在编辑的文件保存在名为filename的文件中
  3. :r filename 用于将 filename文件的内容插入到当前光标位置的后面

第六讲

本讲主要讲各种插入指令

  1. 输入 o 可以在光标下方打开新的一行并且进入插入模式；输入 O 可以在光标上方打开新的一行并且进入插入模式
  2. 输入 a 可以在光标所在位置之后插入文本 (和 i 不同的是 a 的光标会前进一位)；输入 A 可以在光标所在行的末尾插入文本
  3. e 指令可以使光标移到单词的末尾
  4. v 指令进入可视化模式，然后移动光标选中高亮内容后通过 y指令复制高亮内容，然后移动光标使用 p 指令粘贴内容
